Guidebook PDFs for various Pokemon Games

I love Pokemon guidebooks, and I miss them a lot! I know that nowadays it's much easier and commonplace to simply google any information you'd need on your phone, but these guides feel so much more satisfying to use for me.

Since a lot of these can be extraordinarily expensive in the modern day, I thought it would be nice to post the ones I have saved here alongside the Internet Archive. Feel free to save them to your own computer for future use!

Heads up before you start browsing...

  • The files for these guidebooks can be fairly large, and may take a bit of time to load!
  • These will open in the main frame of the website by default, using your browser's default PDF viewer.
  • Since this site was coded and optimized for Firefox, it's recommended - but any browser should work.
  • To open them in a new tab, right click the link and select "open link in new tab" - or hold ctrl and click.
  • Not all browsers have been tested, and as a result you may run into bugs. Let me know if you run into any issues.

If you'd rather view these PDFs from another source, I've uploaded all the ones I have to my page on the Internet Archive. Usually I try to upload them there first, so there may be some that aren't listed here!
