CSS Image Borders of various Pokemon game's textboxes

Recently, I helped out with a stream layout for Willow Fourleafisland's HeartGold Artlocke, using the HG/SS text box borders. A chatter requested that I host some of these borders as css assets for free use, so here we are! Some of these are in use around my site, and some I haven't personally tested before... You'll have to play around with the settings yourself to get them looking how you'd like.

How do I use CSS image borders?

You use these borders by using the CSS border-image property.
